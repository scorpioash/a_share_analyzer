# 📈 极简 A 股智能分析器 (Minimal A-Share Analyzer)

## 📌 这是什么？
这是一个从头构建的纯 A 股量化与诊断助手。
- **动态获取**：输入全拼名字或标的代码，系统会自动无感抓取最新的全维度数据（含当天的复权 K线走势、当前市盈率/市净率、以及滚动大事新闻摘要）。
- **结合经验**：你可以把你的炒股经验和纪律（比如某个形态不买，市盈率多少该跑）写进根目录的 `my_strategy.md` 里。每次调用大模型分析时，AI 都会严格参照你的经验给出评级操作。
- **网页面板**：内置了可视化的 Web 页面，不论在什么设备，都能通过简单的网页输入股票完成诊断并配置大模型 API。
- **兼容主流**：预设了 OpenAI、Claude、Gemini 三大主流标准接口（市面上绝大多数国产大模型如 DeepSeek 等均可直接填入 OpenAI 接口兼容使用）。
  
  > [!TIP]
  > **💡 模型选择建议**：金融市场犹如没有硝烟的战场，盘面研判极其考验 AI 的逻辑推理深度。为了让本系统输出最敏锐、最准确的操作建议，**强烈建议搭配使用 Claude 4.6 或 ChatGPT 5.4 级别的顶级旗舰模型**。
  > 
  > **⚠️ 免责声明**：本系统所有的行情推演及 AI 回复仅仅作为技术的代码测试与个人的逻辑复盘，完全不构成任何投资建议，切莫盲从盲信。股市有风险，交易需谨慎！

## 🚀 怎么使用？

### 方式一：本地电脑启动（推荐新手）
无论你是 Windows 还是 Mac 电脑，只要安装了 Python 3.8+ 环境即可：
1. **Windows**：直接双击运行目录里的 `start.bat`。
2. **Mac / Linux**：打开终端，给满权限并启动：
   ```bash
   chmod +x start.sh
   ./start.sh
   ```
*(注：如果你是生平第一次运行此类程序，黑框中可能会出现 `Email:` 提示让你留邮箱接受订阅，此时**直接敲击一下键盘的回车键(Enter)** 即可跳过！)*

执行完毕后，你的浏览器会自动打开 `http://localhost:8501`。
在页面**侧边栏**填写好你的大模型 API_KEY，即可开始搜股诊断！

### 方式二：Docker 服务器部署
如果你想把它挂在 NAS 或者是云端服务器上，变成 24 小时的私人极简工作站，使用 Docker 拉起是最优解：
```bash
# 后台拉起并暴露 8501 端口
docker-compose up -d
```
启动后，拿手机或电脑浏览器访问 `http://<服务器IP>:8501`。
然后你就可以悠闲地在**网页左侧的侧边栏**填写或者切换你的大模型 API KEY，直接开启诊股即可！
*(注：通过 docker-compose 映射，哪怕你在母机直接用文本编辑器改动了 `my_strategy.md` 保存，网页里的 AI 也会立刻读取新经验生效，无需重启容器！)*

### 方式三：GitHub Actions 定时推送（纯白嫖玩法）
如果你平时没空看网页，希望每天收盘后自动查完发到你的飞书/微信。本库已经写好了自动化脚本，你甚至不需要有自己的服务器：
1. Fork 本仓库。
2. 进入在你名下的仓库 `Settings` -> `Secrets and variables` -> `Actions`。
3. 把你用到的变量填进去（如 `OPENAI_API_KEY`，`FEISHU_WEBHOOK_URL` 飞书机器人链接，`STOCK_LIST`你要跟踪的代码 如 600519）。
4. 搞定！以后的每个交易日下午 16:00，微软的服务器会自动跑一遍诊断发给你！
